# Agent Skills

Skills para Pi e agentes compatíveis.

## Skills

- [`moodle-development`](skills/moodle-development/SKILL.md): decisões corretas para desenvolvimento Moodle core/plugin, APIs, segurança, DB upgrades, UI e testes.

## Instalação no Pi

```bash
pi install git:github.com/BrenoAugustoUEMS/skills
```

Depois, no Pi aberto:

```text
/reload
```

